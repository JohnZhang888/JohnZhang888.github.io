网站重建记

记录我重新搭建本网站的过程。


起因

这几日闲来无事，把之前老掉牙的网站翻修一下。

其实我想重建网站的念头并不是突然冒出来的。在5月份的时候，我注意到好友Rabbit的网站已焕然一新，远超出我的想象（之前我还在自己的网页里吐槽过）。于是我和他吃饭的时候谈起这件事情来，做出承诺，暑假的时候要重建我的网站。

现在暑假已到，该是履行承诺的时候了。

工欲善其事，必先利其器

之前我一直使用 Windows 操作系统编写网站，虽说没有什么不对，但是运行速度慢一点，安装软件麻烦一点，配置环境麻烦一点……这些“一点”逼迫着我离开这个操作系统。对不起，微软，我最终还是选择了 GNU/Linux。至于发行版，我选择了 Linux Mint，基于强大的 Ubuntu 却又没有 snap 和广告的骚扰，运行速度也是杠杠的。

不过微软也不用太伤心，毕竟编辑器还是 Visual Studio Code，从我开始编程的第一天就在用，实在顺手得很。

轮子选得好，不用自己造

看到 Rabbit 跟我炫耀他的网站有多么大，CSS有多么长，我便嗤之以鼻。现在网上现成的 CSS、JS 框架那么多，何必自己造轮子呢？选择自己觉得方便、顺手的一款就行了。于是，我选择了 Bootstrap。

我本来是想用 Tailwind CSS 的，然而大家对比一下下面这两段代码就知道我为什么不用它了。

Tailwind CSS

<button class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded">Button</button>

Bootstrap

<button class="btn btn-primary">Button</button>

开始编写

准备工作完成，接下来就该编写网站了。这个过程比较枯燥：编辑器里写一点，Ctrl + S 一下，然后再切换到浏览器，F5 一下……最后就写好了（其实还没有写好）。


张永康，2023 年 7 月 26 日。
