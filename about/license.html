许可证

我欢迎任何人阅读、修改、转发本网站的内容。但是，你需要遵守一些许可证（也被称为“协议”），来保证本网站的内容不会被非法侵占或胡乱使用。以下是许可证的粗略内容，详细内容请点击相应按钮查询。


代码协议：GNU 公共通用许可证，版本 3.0 或以上。

本网站的所有代码，包括在浏览器中按下 F12 出现的代码、使用 git clone 获取到的代码等，都遵循 GPL v3+ 协议。主要内容如下：

  • 可自由复制：你可以将软件复制到你的电脑，你客户的电脑，或者任何地方。复制份数没有任何限制。
  • 可自由分发：在你的网站提供下载，拷贝到U盘送人，或者将源代码打印出来从窗户扔出去（环保起见，请别这样做；从法律的角度来讲，请别这样做，因为这属于高空抛物。）。
  • 可以用来盈利：你可以在分发软件时候收费，但你必须在收费前向你的客户提供该软件的 GNU GPL 许可协议，以便让他们知道，他们可以从别的渠道免费得到这份软件，以及你收费的理由。
  • 可自由修改：如果你想添加或删除某个功能，没问题，如果你想在别的项目中使用部分代码，也没问题，唯一的要求是，使用了这段代码的项目也必须使用 GPL 协议。

上文中，“本作品”指本网站的所有文本，即可以在浏览器中直接复制到的文本。
